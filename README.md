# SoundLux — Premium Audio E-Commerce Platform

SoundLux is a state-of-the-art e-commerce platform designed for the **INFERNO'26 : HACKOVERFLOW** hackathon. It offers a premium, high-performance shopping experience for audiophiles, featuring a curated selection of world-class headphones and audio equipment.

Developed with a focus on modern aesthetics, real-time data synchronization, and robust security, SoundLux represents the future of specialized retail platforms.

---

## ✨ Key Features

### 🛍️ Dynamic Shopping Experience
- **Real-Time Product Catalog**: Live fetching of high-fidelity audio gear from a centralized Supabase database.
- **Advanced Filtering**: Seamlessly filter by categories like Wireless, Gaming, Studio, and Noise Cancelling.
- **Interactive Product Details**: Deep-dive into specifications, high-res galleries, and verified customer reviews.
- **Smart Search**: Instant search functionality across the entire product ecosystem.

### 🔐 Advanced User Systems
- **Persistent Wishlist & Cart**: Your favorites and selections are synced across devices via Supabase Auth and Database integration.
- **Secure Authentication**: Robust login and signup flows with role-based access control.
- **Verified Reviews**: A community-driven review system where users can share their honest feedback.
- **Order Tracking**: Detailed tracking for every purchase, from processing to delivery.

### 🛡️ Professional Admin Suite
- **Analytics Dashboard**: Real-time visualization of monthly revenue, total orders, and product distribution.
- **Inventory Control**: Full CRUD interface for managing products, including image management and stock tracking.
- **Order Management**: Comprehensive tools for tracking and updating customer order statuses.
- **Marketing Tools**: Integrated coupon and discount code management system to drive sales.
- **User Governance**: View and manage the user base, with the ability to promote/demote administrative roles.

### 🎨 Design & Performance
- **Premium Aesthetics**: A dark-mode first design system featuring glassmorphism, floating 3D-effect visuals, and vibrant neon accents.
- **Fluid Micro-animations**: Smooth transitions and hover effects that make the interface feel alive and responsive.
- **Mobile Perfection**: Fully responsive architecture that provides a consistent premium experience on any screen size.
- **SEO Optimized**: Advanced meta-tagging and OpenGraph support for beautiful link previews on social platforms.

---

## 🛠 Tech Stack

- **Frontend**: [Next.js 14/15+](https://nextjs.org/) (React 19)
- **Styling**: Vanilla CSS (Modern CSS Variables & Grid Systems)
- **Backend / Database**: [Supabase](https://supabase.com/) (PostgreSQL)
- **Authentication**: Supabase Auth
- **State Management**: React Context API + Custom Hooks
